# Netflix SQL Analysis

## Project Overview

This project explores the Netflix Movies and TV Shows dataset using PostgreSQL.

## Tools Used

- PostgreSQL
- pgAdmin 4
- SQL
- GitHub

## Dataset

Netflix Movies and TV Shows dataset from Kaggle.

## Business Questions

1. How many Movies and TV Shows are available?
2. Which countries contribute the most content?
3. What are the most common content ratings?
4. Which years saw the highest content production?

## Key Findings

- Movies represent the majority of Netflix content.
- The United States contributes the largest number of titles.
- TV-MA is among the most common ratings.
